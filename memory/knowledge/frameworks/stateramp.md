# StateRAMP

**Scope**: Cloud services to US state/local government. NIST 800-53 derivative aligned to FedRAMP impact levels.

**Authority**: StateRAMP PMO (501(c)(3)).

**Total controls**: ~125 (Low) / 325 (Moderate) — mirrors FedRAMP baseline with state-specific parameters.

## Mapping

Mirrors [[fedramp]] mappings 1:1 with state-specific overlays. Per-state additions vary (e.g., TX, FL, OK have leading programs).

## Management assertion template

Same as [[fedramp]] with substitution of state PoC + StateRAMP authorization.

## Signing officer

CEO + designated StateRAMP POC. State authorizing official for ATO.

## Audit cycle

Per state authorization + continuous monitoring (similar cadence to FedRAMP).

## Retention

Per state records-retention schedule + 3 years post-decommissioning.
